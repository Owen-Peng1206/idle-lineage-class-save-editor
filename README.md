# 放置天堂 - 存檔編輯器 (Idle Lineage Class Save Editor)

這是一個專為網頁遊戲《放置天堂 - 以血為盟》（Idle Lineage Class）設計的單機版/網頁版存檔編輯器。本專案完全基於前端 HTML 與 JavaScript 實作，無需安裝任何依賴、編譯步驟或後端伺服器，即可在瀏覽器中直接開啟使用。

## 🌟 主要功能

- **多存檔格式支援**：支援讀取與匯出普通 JSON 格式及遊戲預設的 LZ4 壓縮/簽章加密格式（`LZ1:[壓縮內容]:SIG1:[簽章]`）。
- **瀏覽器聯動功能（Bookmarklet）**：提供書籤小工具，可直接讀取並回寫遊戲在瀏覽器 `localStorage` 中的存檔，實現無縫編輯。
- **角色屬性編輯**：
  - 修改角色等級、經驗值、金幣。
  - 自由調整基礎屬性（力量、敏捷、體質、智力、智慧、魅力）、分配點數及萬能藥加成。
  - 調整剩餘自由屬性點。
- **裝備與背包編輯**：
  - 編輯所有裝備欄位（武器、防具、盾牌、飾品等）。
  - 背包與倉庫物品的增刪與編輯，支援設定數量、強化等級、祝福/詛咒狀態、遠古/永恆/不朽/太初等階、詞綴及套裝效果。
  - 提供內建的物品資料庫快速搜尋與加入。
- **技能與進度編輯**：
  - 自由增減角色已學習的技能（與角色職業匹配）。
  - 修改遊戲模式（傳統模式/經典模式）與副本/地下城進度。
  - 編輯圖鑑收集進度（卡片圖鑑、裝備圖鑑、怪物圖鑑、遺物圖鑑）。
- **自動賣出設定**：
  - 在導航列中新增了「自動賣出設定」按鈕（在技能編輯旁邊）。
  - 完整物品資訊顯示：清單中會讀取存檔裡的  junkPrefs  設定，並轉譯出該物品的完整名稱。
  - 每個物品旁邊都有一個「取消自動賣出」按鈕。點擊並確認後，就可以把這個物品從垃圾/自動賣出清單中移除。往後在遊戲中獲得該特定屬性的物品時，就會正常放進背包，不 
  再被系統自動賣掉。

## 🚀 快速開始

1. **下載本專案**：
   ```bash
   git clone --recursive https://github.com/Owen-Peng1206/idle-lineage-class-save-editor.git
   ```
   *(請注意：本專案包含遊戲引擎子模組 `idle-lineage-class`，複製時請加上 `--recursive` 參數以獲取完整的遊戲資料庫與 CSS)*
2. **執行編輯器**：
   - 直接用任何現代瀏覽器（Chrome, Edge, Firefox, Safari）雙擊打開 [save-editor.html](save-editor.html) 即可開始使用。
3. **連動遊戲網頁**：
   - 開啟編輯器，點擊「開啟遊戲網站」或將編輯器中的「書籤小工具」拖曳至瀏覽器書籤列。
   - 在遊戲網頁中點擊該書籤，即可將存檔同步至編輯器；編輯完成後，亦可一鍵同步回寫至遊戲中。

## 📁 專案結構

```

├── save-editor.html          # 編輯器主程式 (單一自足的 HTML/JS 檔案)
├── idle-lineage-class/       # 遊戲引擎子模組 (Git Submodule)
│   ├── index.html           # 遊戲本體入口
│   ├── js/00-data.js        # 遊戲資料庫 (物品、技能、怪物資料)
│   └── css/tailwind-built.css  # 預編譯的 Tailwind CSS 樣式表
├── README.md                 # 本專案說明文檔
├── LICENSE.md                # 本專案授權文檔

```

## 🛠️ 開發與維護

本編輯器為求部署與使用便利，採用**單一 HTML 檔案架構**。所有介面渲染、樣式邏輯、與存檔編解碼均寫在 [save-editor.html](save-editor.html) 中，資料庫依賴idle-lineage-class遊戲本體。

- **樣式調整**：使用預先編譯好的 Tailwind CSS，如需新增或調整客製化樣式，請直接在 [save-editor.html](save-editor.html) 的 `<style>` 區塊內寫入。
- **存檔加密驗證**：專案包含 `_signSave()` 方法，用於生成防篡改驗證雜湊值。修改匯出邏輯時，請確保簽章加密機制正常運作，否則遊戲將無法讀取導出的存檔。

本專案僅供個人學習與交流使用。請勿用於任何商業用途，修改存檔可能會影響遊戲體驗，建議在編輯前備份您的原始存檔。
