歡迎使用存檔編輯器

請載入您的遊戲存檔 JSON 檔案以開始修改屬性、裝備或圖鑑。

✨ 網頁版免部署直接聯動 (最推薦)：

由於瀏覽器安全限制，線上網頁無法直接載入或打開本機 `file:///` 檔案。請依下列步驟進行連線：

  1. 將此按鈕拖曳到您的書籤列（或新增書籤貼上代碼）：
    🔗 傳送存檔至編輯器
  2. 點選下方按鈕打開遊戲網站：

    * 可支援輸入線上網址（如 GitHub Pages）或本機網頁網址（例如 file:///D:/.../index.html）。

  3. 在打開的遊戲網頁中，點擊剛才加入的書籤，存檔就會立刻傳送到這裡！
